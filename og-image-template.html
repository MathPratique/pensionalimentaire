📸 Instructions pour créer l'image OG

  1. Ouvre ce fichier dans Chrome (double-clic sur le fichier)
  2. Appuie sur F12 (DevTools) puis Ctrl+Shift+P
  3. Tape "screenshot" et choisis "Capture node screenshot"
  4. Clique sur l'image bleue ci-dessous (elle apparaîtra sélectionnée dans DevTools)
  5. La capture PNG va se télécharger automatiquement
  6. Renomme le fichier téléchargé en og-image.png
  7. Déplace-le dans le dossier pensionalimentaire/ (racine du projet)
  8. Une fois fait, dis-moi "OG image en place" et je vérifie
